# the new era

something changed in how software gets built

AI-directed engineering — not "AI-assisted coding." My output is no longer lines of code; it's designed systems that produce code correctly and repeatedly.

For eighteen years I was the hands. I wrote the C++ that scaled survivability simulations across thousands of scenarios. I brought up boards by hand — i.MX, STM32, Qualcomm, Zynq — patched the drivers, fought the boot flow, aligned the Hailo and Rockchip accelerator stacks layer by layer until a model finally ran on metal.

The deal has changed. The model can now write the code. What it cannot do is decide what to build, how it should be shaped, where the failure modes hide, and what "done" actually means. That part — the judgment, the architecture, the decomposition, the standard — is now the scarce resource. And it's exactly what eighteen years of low-level, product-grade engineering trained me to do.

So I stopped being only the hands. I became the director. The agents became the hands. And to make that work at the level of quality I demand, I built the cockpit myself.

Every feature, contract, and design constraint in the tooling below was authored by me. The agents wrote the implementation to my specification, under my direction, against my standard — the same way a tech lead directs a team, except the team executes in minutes instead of weeks. That is systems architecture for a workforce of agents — and my embedded discipline (nothing ships unproven) is what keeps the swarm from making a fast, confident mess.

the bench learns

After a mission, a LessonAgent distills what worked into proposed new blocks and skills that flow back into gandalf. Every project I run leaves the workbench better at planning and running the next one.

  • nothing launches blindly — plan → review → approve → launch is enforced law, the same real-time discipline I carried from defense black-box and EW work.
  • the plan is plain markdown — one human-readable dialect every tool speaks; no database hiding the truth.
  • cockpits, not cameras — every running agent is interruptible and injectable; I refuse to lose visibility on a running system.
  • knowledge compounds — lessons become blocks; blocks make the next plan better. My standards became infrastructure.
